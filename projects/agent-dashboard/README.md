Agent Dashboard (Electron + React) prototype

Placement: projects/agent-dashboard

Quick start (once Node.js and npm are installed):
1. cd projects/agent-dashboard
2. npm install --save-dev electron vite react react-dom
3. npm run dev   # starts vite dev server (if configured)
4. npm start     # starts Electron (loads renderer/index.html)

Notes:
- This is a minimal scaffold. Run npm install and wire vite or bundler for a smoother dev experience.
