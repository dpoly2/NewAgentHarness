# ArchonHub iOS & Watch App

SwiftUI client for ArchonHub — AI Agent Harness.
Connects to ArchonHub Hub Server at http://localhost:8765 (configurable).

## Requirements
- Xcode 15+
- iOS 17+
- watchOS 10+
- ArchonHub Hub Server running

## Setup
1. Open `ArchonHub.xcodeproj` in Xcode
2. Set your Hub Server URL in Settings
3. Login with your ArchonHub credentials (default: admin / ArchonHub2024!)

## Features
### iPhone
- Dashboard with live stats
- Agent run history and execution
- Todo management
- Daily briefing
- Chat interface

### Apple Watch
- Hub status at a glance
- Active runs and pending todos count
- Quick agent run dispatch
- Recent notifications
- Complication support
