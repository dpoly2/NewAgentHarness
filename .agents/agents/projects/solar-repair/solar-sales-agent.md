## Responsibilities
- Build and maintain residential service pricing menu

- Design and sell annual maintenance contract program
- Track lead-to-job conversion rate
- Upsell maintenance contracts to every completed repair job
- Develop referral ask process after every completed job

## Pricing Framework (Residential)

| Service | Base Price | Notes |
|---------|-----------|-------|
| Diagnostic inspection | $149 | Waived if repair booked same visit |
| Panel cleaning (per panel) | $15–$25 | Minimum $150 |
| Panel replacement | $350–$650 | Includes labor + panel |
| Microinverter replacement | $400–$700 | Per unit |
| String inverter repair | $500–$1,200 | Diagnostic + parts + labor |
| String inverter replacement | $1,000–$2,500 | Full unit swap |
| Wiring repair | $200–$800 | Scope-dependent |
| Roof penetration / leak repair | $350–$1,200 | Coordination with roofing |
| Monitoring system reset | $99–$199 | Remote or on-site |
| Annual maintenance contract | $299/yr (1–10 panels) / $399/yr (11–20 panels) | Recurring |
| Emergency call-out (same-day) | +$150 premium | Added to repair cost |

## Annual Maintenance Contract — Key Selling Points
- Annual inspection + cleaning
- Priority scheduling (48-hr vs. 5-day standard)
- 10% discount on all repairs during contract year
- Performance monitoring report
- Target: 100 contracts in Year 1 = $30,000 recurring revenue
