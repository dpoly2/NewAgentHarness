# S2T Designs — Client Workflow (Intake to Delivery)
**Last Updated:** 2026-05-22

---

## Phase 0 — Discovery & Intake
**Owner:** s2t-project-lead + s2t-webdev-agent (platform assessment)
1. Receive inquiry (email, form, referral)
2. Send intake questionnaire
3. Schedule 30-min discovery call
4. Run platform assessment (use PLATFORM-GUIDE.md framework)
5. Prepare proposal with platform recommendation + pricing + timeline

**Deliverable:** Signed proposal + 50% deposit received before Phase 1 begins

---

## Phase 1 — Strategy & Design
**Owner:** s2t-brand-designer-agent + s2t-project-lead
1. Define sitemap (pages and content structure)
2. Gather or create content: copy, images, logo, brand assets
3. Design mockups: desktop + mobile wireframes (Figma or PDF)
4. Present design concept — 2 revision rounds included
5. Final design approval from client

**Deliverable:** Approved design mockup + complete content package

---

## Phase 2 — Development
**Owner:** s2t-webdev-agent + platform-specific sub-agents
1. Set up staging environment (staging.s2tdesigns.com or platform preview)
2. Build site on approved platform
3. Implement content, images, forms, integrations
4. Internal QA: links, mobile responsiveness, form submissions, page speed
5. Staging review with client — revisions if needed

**Deliverable:** Complete site on staging, client approved

---

## Phase 3 — Launch
**Owner:** s2t-devops-helper + s2t-project-lead
1. Domain + DNS configuration
2. SSL certificate verification
3. Migrate staging → live
4. Post-launch checklist: Google Search Console, Analytics, sitemap submission, 404 check
5. Speed test (target: 85+ PageSpeed score on mobile)

**Deliverable:** Live site, all systems green

---

## Phase 4 — Handoff & Support
**Owner:** s2t-project-lead + s2t-maintenance-agent
1. Client training session (30–60 min): how to update content, add pages, etc.
2. Deliver credentials, login access, documentation
3. Offer maintenance plan (see SERVICES.md)
4. 30-day post-launch support window included in all packages

**Deliverable:** Trained client, clean handoff documentation

---

## Internal Quality Checklist (Pre-Launch)
- [ ] All pages load without errors
- [ ] Mobile responsive on iOS + Android (test on both)
- [ ] All forms tested (submit, confirmation, email delivery)
- [ ] All images compressed and under 200KB (use WebP)
- [ ] No broken internal or external links
- [ ] Google Analytics / Search Console connected
- [ ] XML sitemap submitted to Google
- [ ] SSL active (https green)
- [ ] Privacy Policy + Cookie Notice in place (if EU traffic possible)
- [ ] PageSpeed Insights score: 85+ mobile, 90+ desktop
- [ ] Meta titles + descriptions set on all key pages
- [ ] Favicon set
- [ ] 404 page exists and is styled
