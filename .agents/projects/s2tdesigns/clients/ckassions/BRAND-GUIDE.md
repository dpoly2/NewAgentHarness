### Design Placements
| Design | Placement | Technique |
|--------|-----------|-----------|
| Primary Logo | Left chest | Embroidery (hats) / Screen print (tees) |
| Full graphic | Center chest / Back | DTG or Screen print |
| Wordmark | Sleeve / Back neck | Screen print or woven label |

### File Specs for Print
- **Vector:** .AI or .EPS (preferred for all logo placements)
- **Raster:** 300 DPI minimum, .PSD or .PNG with transparency
- **Screen Print:** Separated files, max 4 spot colors per design
- **DTG / POD:** 300 DPI PNG with transparent background
- **Embroidery:** DST or EMB file (converted from vector) — 15,000–20,000 stitch count for chest logo
