## Brand Identity Overview

CKassions is a luxury lifestyle and apparel brand built on elegance, femininity, and magic. The brand's visual identity communicates effortless sophistication — gold as the primary color signals premium quality, while the Tinker Bell accent adds a whimsical, aspirational dimension that sets the brand apart from standard luxury streetwear.

### Brand Personality
**Elegant. Magical. Unapologetically feminine. Premium.**

Think: luxury lifestyle meets fairy-tale confidence. The woman who wears CKassions doesn't follow trends — she sets them.

### Target Audience
Stylish, aspirational women (and men who gift them). Fashion-forward consumers who value quality, elegance, and brands that feel personal and intentional. Age range: 20s–40s. Social media-driven, aesthetics-conscious.

**Use cases for standalone CK mark:**
- Instagram profile photo
- Favicon (browser tab)
- Embossed leather labels on premium items
- Wax seals on packaging
- Watermarks on photography
- Hat embroidery (chest mark too detailed for small hat crowns)

## Logo Usage Rules

### ✅ Approved Usage
- Gold logo on white, cream, or very light backgrounds
- Gold logo reverse (white logo) on deep jewel tones: navy, black, deep burgundy, forest green
- Black logo on white/cream/natural backgrounds only
- Signature (Tinker Bell) logo for hero/feature usage only — not repeated patterns

### ❌ Prohibited Usage
- Do not place the logo on busy photographs without a clean overlay or background shape
- Do not stretch, compress, rotate, or distort any logo version
- Do not change the gold color to any other metallic or yellow shade without approval
- Do not recolor the black version to gray — use full black only
- Do not add drop shadows, glows, or effects to the logo
- Do not combine Logo 1 and Logo 3 in the same layout
- Do not use pixelated or low-resolution exports for any print purpose

### Minimum Size
- Digital: Logo should never be rendered smaller than 120px wide
- Print: Logo should never be smaller than 1.25 inches wide
- Standalone CK mark: 80px minimum digital / 0.75 inch minimum print

### Clear Space
Maintain a clear space equal to the height of the "K" in the monogram on all four sides of the logo. No text, graphics, or other elements within this zone.

## Apparel Application Guide

### Garment Colorways
| Garment Color | Logo Version to Use | Application Method |
|--------------|--------------------|--------------------|
| White / Cream / Natural | Black Logo (Logo 2) | Screen print / DTG |
| Black / Charcoal / Navy | Gold Logo (Logo 1) | Screen print / DTG / Embroidery (gold thread) |
| Gold / Camel / Tan | Black Logo (Logo 2) | Screen print |
| Burgundy / Deep Red | Gold Logo (Logo 1) | Screen print / Embroidery |
| Gray | Either — test both | Screen print |

### Recommended Apparel Items & Placements

| Item | Placement | Logo Version | Method |
|------|-----------|-------------|--------|
| T-Shirt (unisex/women's) | Left chest | Logo 1 or 2 | DTG or Screen print |
| T-Shirt (graphic tee) | Center chest, full size | Logo 3 (Signature) | DTG |
| Hoodie | Left chest + back neck | Logo 1 or 2 | Screen print |
| Snapback / Dad Hat | Front crown | CK Monogram only | Embroidery |
| Tote Bag | Center | Logo 3 (Signature) | Screen print or DTG |
| Joggers | Left hip | CK Monogram | Embroidery or heat transfer |
| Packaging / Tissue | Center | Logo 1 (Gold) | Gold foil stamp or print |
| Hang Tag | Front + back | Logo 2 (Black) | Digital print or letterpress |

### Print File Specs
- **Screen Print:** Vector EPS or AI, separated layers, max 4 spot colors, PMS referenced
- **DTG / POD (Printful/Printify):** 300 DPI minimum, PNG with transparent background, sRGB color profile
- **Embroidery:** DST file required (convert from vector), ~10,000–20,000 stitch count for chest logo
- **Hang Tag:** 300 DPI PDF, bleed: 0.125 inch all sides, CMYK color mode
