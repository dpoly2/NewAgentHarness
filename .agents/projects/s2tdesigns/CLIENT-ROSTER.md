## Client Intake Template
When a new prospect contacts S2T Designs, collect:
1. Business name + website (if any)
2. Project goal (new site / redesign / e-commerce / branding / maintenance)
3. Technical ability (none / basic / intermediate / developer)
4. Budget range
5. Timeline / deadline
6. Competitor sites they like (for style reference)
7. Content readiness (copy + images ready or need help?)

→ Send to s2t-project-lead to kick off platform assessment and proposal
