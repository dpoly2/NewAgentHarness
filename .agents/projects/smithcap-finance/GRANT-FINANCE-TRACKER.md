## FINANCIAL COMPLIANCE RULES BY FUNDER TYPE

### Federal Grants (HUD, EDA, FHWA)
| Rule | Requirement |
|------|------------|
| Procurement | Competitive bidding required for contracts >$10K |
| Documentation | Every expense must have invoice + cancelled check + proof of work |
| Reporting | Semi-annual or annual financial and narrative reports |
| Audit | Single Audit required if federal awards >$750K in a year |
| Matching | Many federal grants require 20–50% local match |
| Davis-Bacon Act | Federal prevailing wage rates for construction workers |
| DUNS/SAM | Must be registered in SAM.gov before receiving federal funds |

### Private/Foundation Grants
| Rule | Requirement |
|------|------------|
| Restricted use | Funds must be used for stated purpose only |
| No self-dealing | Funds cannot benefit the grantee's officers/directors personally |
| Reporting | Typically semi-annual or annual narrative + financial report |
| Audit | Some funders require independent audit if award >$100K |
| Logo/attribution | Most require funder acknowledgment in all materials |

### Nonprofit-Specific (XFTC & PBS)
| Rule | Requirement |
|------|------------|
| Restricted vs. unrestricted | Track separately in Wave — never co-mingle |
| Board approval | All grants >$25K should have board resolution on file |
| Public disclosure | Form 990 Schedule B discloses major donors (>$5K) |
| UBIT check | If grant funds activities unrelated to mission, UBIT may apply |

The `grants-research-agent` runs every Monday at 8:00 AM CT and scans for:

- Grants.gov (federal)
