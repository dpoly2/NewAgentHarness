## Build Options

### Option A — Fastest (Today)
Use **Carrd.co** — single page site builder, free tier available
- Drag and drop, mobile responsive
- Custom domain: point theElevationATX.com → Carrd
- Has built-in form with email capture
- Live in under 2 hours

### Option B — More Control
Use **Shopify Landing Page** (already have account via Hoodswag)
- Add a new page to hoodswag.shop/pages/the-elevation
- Redirect theElevationATX.com → that page
- Form submissions go to email

### Option C — Full Custom (Best Long Term)
Build as a WordPress page on a new domain
- Same stack as xtremeforcetrackclub.org
- Full control over design and data
- Waitlist stored in database
- Timeline: 1–2 days

**Recommendation:** Start with Carrd today to capture demand immediately. Migrate to custom WordPress once Event 1 is confirmed.
